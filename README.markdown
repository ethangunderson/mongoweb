Mongo_Web
========

Mongo_Web is the stylish way of inspecting MongoDB databases.

Installing
----------

gem install mongo_web

Usage
-----

Starting MongoWeb is simple

    $ mongo-web

You can also pass it a specific port

    $ mongo-web -p 6969
